Settings menu

Pulldown choices:

  • Accounts in separate OFX files
    Not yet correctly implemented, default is that all transactions are divide according to account.
    If NOT checked then the result is unpredictable.
  • Only interest transactions
    This is "old" functionality, it filters ING savings transactions, only the interest transactions are stored.
  • GnuCash executable
    Points to the location of the GnuCash executable, defaults to C:\Program Files (x86)\gnucash\bin\gnucash.exe.
  • Synonym file

    Points to a synonym file.

    A prefix can be configured, in case there are multiple accounts.

    See Synonym configuration.

  • Loglevel
    For debugging purposes, the detail level of the messages can be adjusted(Normal level is INFO).
  • Look and Feel
    The look and feel of the application windows can be choosen.
  • Create logfiles
    A directory where the logfiles are created is choosen.
