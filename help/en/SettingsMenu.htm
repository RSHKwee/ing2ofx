Settings menu

Pulldown choices

  • Only interest transactions

    This is "old" functionality, it filters ING savings transactions, only the interest transactions are stored.

    The transactions are now paired, in this way the filtering takes place.

  • GnuCash executable
    This points to the location of the GnuCash executable, and defaults to C:\Program Files (x86)\gnucash\bin\gnucash.exe.
  • Synonym file

    This points to a synonym file.

    A prefix can be configured, in case there are multiple accounts. See Synonym configuration.

  • Loglevel
    For debugging purposes, the detail level of the messages can be adjusted (normal level is INFO).
  • Look and Feel
    The look and feel of the application Windows can be chosen.
  • Create logfiles
    A directory where the log files are created can be chosen.
