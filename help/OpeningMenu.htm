Opening menu

Buttons:

  • CSV/XML File

    With this button the input file(s) (csv or xml) is/are selected. With the ctrl-key multiple files can be selected.

    The application recognizes:

    • a comma or semicolon separated ING input file.
    • a SNS XML input file.

    Checkbox Clear transactions: If checked, then the cache is cleared before the file(s) are read otherwise the transactions are added to the previously stored transactions.

    • Read transactions

      The transactions are read and stored in the cache.

  • Output folder

    Points to the directory where the generated OFX-file(s) are stored. A proposal for the "Output filename" is made, this can be changed.

    • Convert to OFX
      The conversion to OFX format is started, the progres is shown in the lower panel.
  • Start GnuCash: GnuCash is started.
