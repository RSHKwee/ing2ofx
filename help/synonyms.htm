Synonyms

An adminstration may not limited to one bank account. 
An input file may contain transactions for multiple accounts / administrations

So a mechanism is available to determine for which account an OFX-file is meant.
It is confgured with a csv-file the file has the following format:


      Seq; AccountNr        ; Prefix
       1 ; NLIyyINGBxxxxxxx ; Monkey
       2 ; NLzzSNSBnnnnnnn  ; Nuts
       ......

This results in the following filenames (see Filename convention):

  
   Monkey_NLIyyINGBxxxxxxx_<InputFilename>.ofx
   Nuts_NLzzSNSBnnnnnnn_<InputFilename>.ofx
The prefix may point to the Administration where the account is administered.
