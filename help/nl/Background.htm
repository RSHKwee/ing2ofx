Achtergrond

Het doel van deze (java) applicatie is om ING (ING site) csv- of SNS (SNS Bank site) xml-bestanden te converteren naar OFX-bestanden die door GnuCash (www.gnucash.org) kunnen worden gelezen.

Dit programma heeft als basis een Python script, zie GitHub: https://github.com/chmistry/ing2ofx/releases. Dit script heeft wijzigingen nodig om compitabel te zijn met de nieuw(ste) Python versie en wijzigingen het ING CSV formaat.

Het installeren van de nieuwe Python versie leverde onoverkomelijke (versie) problemen op. Het Python script is herschreven in Java.

De OFX Specificatie kan worden gedownloaded van OFX.NET

Een handleiding over hoe je een admistratie in GnuCash opzet, is te vinden op Beginnen met boekhouden in GnuCash.
