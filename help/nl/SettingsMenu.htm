Instellingen menu

Vervolgkeuzes

  • Alleen rente transacties

    Dit is "oude" functionaliteit, het filters in het ING savings transacties bestand, de rente transacties.

    De nieuwe functionaliteit is dat de transacties nu worden gepaard, op deze manier vindt filtering plaatst.

  • GnuCash executable
    Verwijzing naar de loactie van de GnuCash executable, standaard wordt verwezen naar C:\Program Files (x86)\gnucash\bin\gnucash.exe.
  • Synonymenbestand

    This points to a synonym file.

    A prefix can be configured, in case there are multiple accounts. See Synonym configuration.

  • Loglevel
    Voor debugging doeleinden, gedetailleerde messages kan worden ingesteld (standaard is het detail niveau INFO).
  • Uiterlijk
    Het uiterlijk van de Windows kan worden gewijzigd.
  • Creeer logbestanden
    Verwijst naar de directory waar de log-bestanden worden gecreeerd.
